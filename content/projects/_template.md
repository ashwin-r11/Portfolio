## How to add a project

1. Create a new `.md` file in `content/projects/`
2. Add your thumbnail image to `/public/` folder
3. Update the `image` field to point to it (e.g., `/my-project.jpg`)
4. Write your documentation below
5. Commit and push to deploy!
